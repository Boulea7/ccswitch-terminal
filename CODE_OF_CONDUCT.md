# Code of Conduct

## Our Standard

This project aims to keep discussion direct, respectful, and useful.

Examples of expected behavior:

- assume good intent
- give technical feedback without personal attacks
- include evidence when disagreeing
- keep bug reports and reviews focused on the work
- help newer contributors get unstuck

Examples of unacceptable behavior:

- harassment or insulting language
- doxxing or posting private credentials
- repeated bad-faith arguments
- deliberately disruptive issue or review behavior

## Enforcement

Project maintainers may edit, hide, lock, or remove content that violates these expectations, and may limit further participation if needed.

## Reporting

If you encounter unacceptable behavior, report it privately where possible and include links, screenshots, or other evidence with sensitive details removed.
